# 1. 路线目标

本路线用于验证以下体验：

- 玩家能否在一局内形成“守剑”或“游剑”风格；
- 新招式能否改变护送、突围与终局决斗；
- 玩家是否会在受托、护人、师门和自保之间真正犹豫；
- 秘籍、装备和信物是否带来行动方式，而不只是数值加成；
- 玩家完成后是否愿意改走另一条武学路线，或进入任朔视角。

单局预计经历 8～12 个节点。D04A 与 D04B 互斥；若在 D08 完全履行命令会提前结算，其余路线进入完整终局。

# 2. 开局状态

## 2.1 人物状态

| 状态 | 初始值 | 说明 |
|---|---:|---|
| 伤势 | 无伤 | 依次为无伤、轻伤、重伤、濒危 |
| 内息 | 3 | 进阶招式和强行行动会消耗 |
| 余时 | 3 | 代表城破前可自由支配的时间段 |
| 风声 | 起疑 | 依次为平静、起疑、暴露、缉拿 |
| 人情 | 1 | 可用于报信、求医或请人掩护，只能动用一次 |
| 许简信任 | 1 | 影响底稿、证词和托付 |
| 阿萝信任 | 0 | 影响治疗、藏路和是否同行 |
| 赵衡关系 | 同门 | 后续可能变为信任、猜忌、决裂 |

## 2.2 初始装备

- 兵器：师门短剑；
- 衣甲：百姓夹衣；
- 信物：师门剑印；
- 消耗品：止血散 × 1，火折 × 1；
- 剧情物品：封存文书。

## 2.3 关键状态标记

| 标记 | 含义 |
|---|---|
| `xu_safe` | 许简仍可行动并参与终局 |
| `xu_truth` | 杜缄已知赵衡改动副本 |
| `original_leaf` | 持有带原始分类的残页 |
| `aluo_with_player` | 阿萝选择与杜缄同行 |
| `civilians_follow` | 病患和军属进入护送队伍 |
| `zhao_promise` | 杜缄曾向赵衡承诺交人或交文书 |
| `ren_trace` | 发现任朔或并州斥候留下的痕迹 |
| `manual_half` | 杜缄当前持有《折锋六势》残谱 |
| `guard_style` | 本局进入守剑路线 |
| `roam_style` | 本局进入游剑路线 |
| `guard_art` | 已悟“横剑截流” |
| `roam_art` | 已悟“踏墙换位” |
| `disarm_art` | 已得隐藏招式“卸锋” |
| `zhao_shoulder_known` | 从任朔周目得知赵衡左肩旧伤 |

# 3. 路线总览

```text
D01 师兄授命
  ↓
D02 西市伏袭（第一次交锋、积累武学倾向）
  ↓
D03 伤者与持卷者（第一次不可兼得）
  ├─ 留下救许简 → D04A 旧宅真言
  └─ 追击持卷者 → D04B 屋脊断箭
  ↓
D05 军属里巷（阿萝与护送目标）
  ↓
D06 一甲一药（装备与救治取舍）
  ↓
D07 废寺残谱（定路、成招）
  ↓
D08 师兄索人（不可逆立场）
  ↓
D09 巷口三刃（验证新招式）
  ↓
D10 城门之前（决定谁先走）
  ↓
D11 残照对锋（赵衡终局交锋）
  ↓
D12 留人、留剑或留名（结局选择）
```

# 4. 事件节点

## D01 师兄授命

**地点**：王允府外巷  
**目的**：建立任务、师门关系和第一次信息取舍。

### 场景文本

> 府门内的灯还亮着，门外却已换了第三班持戟武吏。赵衡没有披甲，只把一卷封泥未干的文书递到你手中。
>
> “护许简去城西旧宅，再把誊录名册的医女带回来。”
>
> 他说“带回来”时，右手仍压在刀柄上。那是师父生前纠正他最多的一处旧习。

### 选项

| 选项 | 可见提示 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【接过文书，不再多问】 | 保住时机与师兄信任，但你不知道阿萝为何必须回来 | 赵衡关系偏信任；余时不变 | D08 更容易交涉；无法提前识别赵衡命令中的异常 |
| 【问清阿萝犯了什么罪】 | 花时间追问，可能引起师兄猜忌 | 余时 -1；获得信息“阿萝是证人，不是犯人”；赵衡关系偏猜忌 | D05 可直接指出任务矛盾；D08 赵衡更早防备 |
| 【借整理衣襟，验看封泥】 | 需要谋；不拆封，只判断文书是否被重封 | 发现封泥有二次加热痕迹；风声仍为起疑 | D04A 更容易相信许简；D08 可质问文书来源 |
| 【先托府中旧识给养母报信】 | 消耗一次人情，确保她提前避难 | 人情耗尽；养母获得安全标记 | 后续无法用人情求医或请人掩护 |

### 收束

所有选项进入 D02。玩家无法在这里直接拒绝任务，否则缺少推动力；真正的背离必须建立在亲历之后。

## D02 西市伏袭

**地点**：西市废仓外  
**类型**：普通交锋，1～2 轮  
**目的**：第一次以行动积累守剑或游剑倾向。

### 场景文本

> 废仓门轴才响半声，一枚短箭便钉进许简身后的木柱。两名蒙面人从麻包后扑出，一人直取文书，一人横刀封住退路。
>
> 许简不会武。他下意识抱住文书，反而把自己的胸口让在刀锋之前。

### 选项

| 选项 | 条件 | 即时效果 | 成长与后果 |
|---|---|---|---|
| 【横剑挡在许简身前】 | 师门短剑 | 消耗内息 1；许简免受重创 | 守剑领悟 +1；持卷者更容易逃脱 |
| 【踏上木箱，越过持刀人追箭手】 | 无重伤；未穿札甲 | 许简受轻伤；你逼退箭手并看见并州旧箭簇 | 游剑领悟 +1；获得 `ren_trace` |
| 【割断货架绳索，分开两名敌人】 | 谋或辨迹 | 火油罐翻落，战斗立即结束；火折可能受潮 | 守剑与游剑均不增长；D03 多出“先处理火油”风险 |
| 【亮出师门剑印喝问来路】 | 师门剑印 | 对方迟疑，你保住许简但身份被认出 | 风声转为暴露；赵衡之后更快找到你 |

### 反馈要点

- 横剑不能只是“格挡成功”，要写出窄门、距离和许简在身后的负担；
- 踏箱追敌必须明确玩家放弃了许简身边的位置；
- 战斗不产生死亡，先让玩家理解“赢”与“护住目标”不是一回事。

## D03 伤者与持卷者

**地点**：废仓内外  
**目的**：第一次明确要求玩家放弃一个目标。

### 场景文本

> 一名袭击者夺走副本，翻上后墙。许简靠着门框缓缓坐下，指缝里的血已经浸透袖口。
>
> 墙外脚步渐远。屋内火油顺着砖缝爬向灯脚。你最多只能先做一件事。

### 选项

| 选项 | 可见代价 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【留下替许简止血】 | 放走持卷者，消耗止血散或余时 | 使用止血散：`xu_safe`；不用药则余时 -1、自行压迫止血 | 进入 D04A；许简有机会交出原始残页 |
| 【越墙追赶持卷者】 | 许简伤势可能恶化 | 游剑领悟 +1；夺回被改动副本 | 进入 D04B；许简默认重伤，若无人返回相救则终局不能行动 |
| 【先踢翻灯脚，拖许简出仓】 | 文书和持卷者都可能失去 | 保住许简；封存文书被火燎去一角；余时 -1 | 进入 D04A；获得“烧损文书”，可在 D12 用于留录但证据不完整 |
| 【呼喊巡卒，自己继续追】 | 风声将转为暴露 | 巡卒救下许简；你继续追赶 | 进入 D04B；`xu_safe`，但赵衡提前得知你改变路线 |

## D04A 旧宅真言

**触发**：D03 留下救许简  
**地点**：王允府旧宅  
**目的**：给予足够信息，但不把剧情变成查案。

### 场景文本

> 旧宅的窗纸早被风揭去。许简从贴身衣层中取出一页薄简，上面的人名旁有三种不同墨记。
>
> “赵衡删了这些记号。”他说，“我不知道他会杀多少人。但阿萝知道原簿是什么样子，她不能落到他手里。”

### 选项

| 选项 | 可见代价 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【收下原始残页，答应先护阿萝】 | 正式偏离原命令 | 获得 `original_leaf`、`xu_truth`；许简信任 +1 | D08 可用残页质问赵衡；赵衡关系更易决裂 |
| 【只护许简，不碰残页】 | 维持任务边界，但放弃关键证据 | `xu_safe`；许简信任 -1 | D12 无法达成完整“残剑留录”；更容易获得守令结局 |
| 【让许简带残页自行藏匿】 | 你不携带证据，但许简独自行动有风险 | `xu_truth`；余时不减 | 若 D10 不安排许简先走，他更可能死于城乱 |

### 收束

进入 D05。此节点最多说明“赵衡改了名册”，不要求玩家继续寻找凶手。

---

## D04B 屋脊断箭

**触发**：D03 追赶持卷者  
**地点**：西市屋脊  
**类型**：短交锋  
**目的**：提供任朔路线交叉痕迹和游剑机会。

### 场景文本

> 持卷者跃过第二道屋脊时脚下一滑。一支箭从更远处飞来，没有射你，而是射断他用来落脚的旗绳。
>
> 那人摔进草棚，怀中的副本散落一地。远处弓手已经收弓，只在瓦上留下半枚带血的靴印。

### 选项

| 选项 | 条件 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【先夺副本，不管摔伤的人】 | 无 | 获得改动副本；余时不减 | 阿萝初见时信任 -1；知道文书有异但不知原貌 |
| 【救下摔伤者，再问谁放的箭】 | 消耗余时 1 | 获得 `ren_trace`；得知“并州斥候也在找阿萝” | 任朔周目解锁杜缄曾救人的记录；阿萝信任更易提升 |
| 【沿血脚印追远处弓手】 | 游剑领悟至少 1 | 游剑领悟 +1；失去副本 | 解锁任朔路线入口提示；D05 只能凭口信说服阿萝 |
| 【杀死持卷者灭口】 | 师门短剑 | 风声转为暴露；获得副本 | 关闭“不杀而胜”标签；赵衡会认可你的果断但阿萝敌视 |

### 收束

进入 D05。远处弓手身份在杜缄路线不确认，只作为另一周目的邀请。

## D05 军属里巷

**地点**：城西军属里巷  
**目的**：把任务从“带走证人”转为“尊重一个有自主目标的人”。

### 场景文本

> 阿萝没有藏。她正跪在院里替一名凉州老妇缝合腿伤，四周还躺着三个不能独自行走的人。
>
> “我知道你为何来。”她没有抬头，“要我走，可以。先告诉我，这些人怎么办。”

### 选项

| 选项 | 可见代价 | 即时效果 | 成长与后果 |
|---|---|---|---|
| 【答应把伤者一同送出城】 | 队伍变慢，余时 -1 | `aluo_with_player`、`civilians_follow`；阿萝信任 +2 | 守剑领悟 +1；D09 追兵增加，护人结局开放 |
| 【只带阿萝和她弟弟】 | 保住速度，必须放弃病患 | `aluo_with_player`；阿萝信任不增加 | D06 阿萝拒绝优先为你疗伤；终局更容易保存证据 |
| 【告诉她原路和口令，由她自己决定】 | 你放弃控制她，风声可能因引开追兵而趋紧 | 阿萝信任 +1；你获得游剑领悟 +1 | 阿萝暂不同行；D09 根据风声决定她能否自行抵达 |
| 【以王允府命令强行带人】 | 不耗时间，但会彻底破坏信任 | `aluo_with_player`；阿萝信任 -2；赵衡关系偏信任 | 她不会治疗你，也可能在 D10 设法离队 |

## D06 一甲一药

**地点**：军属里巷后院  
**目的**：让装备与消耗品产生剧情代价。

### 场景文本

> 墙角靠着一副从死人身上剥下来的破损札甲。阿萝只剩一包止血散，院里伤得最重的老人已经开始发冷。
>
> 远处传来第一次闭门鼓。你若穿甲，轻身法便难以施展；若不留药，下一场刀剑不会给你喘息的机会。

### 选项

| 选项 | 条件 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【换上破损札甲，把夹衣留给伤者】 | 衣甲槽可用 | 获得札甲；可抵挡一次重击 | 风声至少为暴露；游剑成招选项关闭，直到弃甲 |
| 【保留夹衣，把札甲绑在担架外侧】 | `civilians_follow` | 队伍第一次遭箭时免于减员 | 自身无防护；守剑领悟 +1 |
| 【把止血散留给自己】 | 尚有止血散 | 消耗品保留至自己受伤时自动使用 | 老人死亡；阿萝信任 -1 |
| 【让阿萝救老人】 | 阿萝信任不为负 | 消耗止血散；老人存活 | 阿萝交出《调息图诀》残篇位置；自己后续负伤只能靠内息压制 |
| 【立即出发，甲与药都不动】 | 无 | 余时不减 | 装备与药留在原地；任朔路线可能发现它们 |

## D07 废寺残谱

**地点**：废寺偏殿  
**目的**：第二成长节点；玩家必须在练功、赶路和人物关系间取舍。

### 场景文本

> 师父的旧剑匣藏在佛座后，只剩半本《折锋六势》。匣盖内侧刻着一行小字：守一人易，知所守难。
>
> 城门方向又响了一遍鼓。你有时间参悟一势，也可以把这点时间留给活人。

### 选项

| 选项 | 条件 | 代价 | 解锁与后果 |
|---|---|---|---|
| 【以门框作界，演练守势】 | 守剑领悟至少 1 | 余时 -1 | 设置 `guard_style`；解锁 `guard_art`“横剑截流” |
| 【沿断墙与残梁试走身位】 | 游剑领悟至少 1；未穿札甲 | 余时 -1 | 设置 `roam_style`；解锁 `roam_art`“踏墙换位” |
| 【不练，把残谱交给阿萝保存】 | 阿萝同行且信任至少 1 | 放弃本局成招 | 获得 `manual_half` 的传承标记；结局可得“残谱传人”，下周目开放起始路线 |
| 【不停留，趁闭门前赶路】 | 无 | 放弃残谱 | 余时不减；D10 路况较好，但终局只能使用基础剑法 |
| 【按《调息图诀》压住暗伤】 | 已知残篇位置或阿萝信任至少 2 | 内息上限临时 +1；余时 -1 | 终局可多支撑一轮；结算留下内伤风险 |

### 设计规则

- 选择前两项参悟时，杜缄将把残谱带走并设置 `manual_half`；选择托付则记录传承，但杜缄本局不持有残谱；
- 玩家在 D02、D03、D05 的行为决定这里哪些路线可学，不使用菜单式自由加点；
- 如果两种领悟都满足，玩家仍只能选一种；
- 放弃练功不是错误路线，能换取更早抵达城门。

## D08 师兄索人

**地点**：废寺山门或通往城门的窄巷  
**目的**：不可逆立场选择。

### 场景文本

> 赵衡没有带大队人马，只带了两名持弩武吏。他站在石阶下，看了一眼你身后的人，又看了一眼你的剑。
>
> “师弟，把医女和文书给我。今日之事，我当你从未走错路。”

### 选项

| 选项 | 条件 | 即时效果 | 延迟后果 |
|---|---|---|---|
| 【履行命令，交出阿萝和文书】 | 阿萝同行 | 设置 `zhao_promise` 已履行；赵衡关系信任 | 跳过 D09～D11，直接进入 D12 的守令结算；阿萝命运危险 |
| 【只交文书，要求放人】 | 持有封存文书或副本 | 失去文书；赵衡暂时退让 | 阿萝同行；D11 赵衡仍会索取原始残页 |
| 【亮出原始残页，质问他删去分类】 | `original_leaf` 或 `xu_truth` | 赵衡关系决裂；风声转为暴露 | 获得他的回答：“乱世没有逐一审人的时间”；D11 对决不可避免 |
| 【以同门身份承诺稍后交人，先求一条生路】 | 师门剑印；赵衡非决裂 | 设置 `zhao_promise`；获得余时 +1 | D11 若背约，赵衡攻击更狠；若守约，可能获得“卸锋”指点 |
| 【拔剑拒绝】 | 无 | 赵衡关系决裂；进入 D09 | 若尚未定路，系统根据此前领悟较高者临时定路，但无法获得成招 |

## D09 巷口三刃

**地点**：军属里巷出口  
**类型**：第二次交锋，1～3 轮  
**目的**：让刚学会的武学立即改变行动。

### 场景文本

> 三名追兵堵住巷口。前面两人持刀，第三人已经踩上矮墙，想从屋顶越过你去抓阿萝。
>
> 巷宽不足一丈。你可以守住地面，也可以抢先登墙，但不能同时站在两个位置。

### 选项

| 选项 | 条件 | 代价 | 结果 |
|---|---|---|---|
| 【横剑截流，守住这一丈】 | `guard_art` | 内息 -1；若无札甲则伤势至少轻伤 | 护送队伍无减员；解锁终局“一人当巷”用法 |
| 【踏墙换位，先制住屋顶追兵】 | `roam_art`；未穿札甲 | 风声转为暴露；地面追兵逼近阿萝 | 可夺取弩箭并绕到敌后；若阿萝信任高，她会组织病患自行退入侧巷 |
| 【以剑印喝止同门武吏】 | 师门剑印；赵衡关系非决裂 | 风声转为缉拿 | 不战通过，但赵衡知道你的准确位置，D11 无法避开 |
| 【割断棚架，封住巷口】 | 师门短剑 | 短剑受损；余时 -1 | 无需受伤通过；终局无法使用需要完整兵刃的“卸锋” |
| 【丢下文书，引追兵争抢】 | 持有文书或残页 | 永久失去所丢物 | 队伍安全通过；相应史证与留录结局关闭 |

## D10 城门之前

**地点**：西侧城门附近  
**目的**：决定谁先走，形成结局的主要人命代价。

### 场景文本

> 城门只开着容一辆车通过的缝。败兵、军属和百姓挤在拒马之间，守卒已经开始驱赶没有路引的人。
>
> 许简要保住底稿，阿萝要带走伤者。你若留下引开追兵，自己未必还能赶上城门关闭。

### 选项

| 选项 | 条件 | 代价 | 结果 |
|---|---|---|---|
| 【让阿萝和伤者先走，自己留下断后】 | 阿萝同行 | 自己进入 D11；余时视为 0 | 阿萝及病患获得高生还率；开启“负伤断后”与“负刃护人” |
| 【护许简和底稿先过拒马】 | `xu_safe` | 阿萝和军属留在原地 | 史证容易保存；阿萝命运取决于她的信任与是否掌握藏路 |
| 【分成两队，自己在中间接应】 | 余时至少 1；风声低于缉拿 | 内息 -1；两队都有受袭风险 | 若 D09 无减员，两队均可到达门边；否则随机性改为条件判定，不能无提示抽签 |
| 【把剑印交给阿萝，让她冒充府中人】 | 师门剑印；阿萝信任至少 1 | 失去信物 | 阿萝一队先过；杜缄无法在 D11 以同门身份交涉 |
| 【独自携文书过门】 | 持有文书 | 放弃所有同行者 | 快速进入守令结局；关闭护人类标签 |

## D11 残照对锋

**地点**：城门内侧  
**类型**：终局交锋，3～5 轮的压缩节点  
**目的**：让武学、装备、关系和前序承诺共同决定终局方法。

### 场景文本

> 夕光贴着城墙落下来，把赵衡的影子拖得很长。他没有再带弩手。远处已经传来城中军阵崩散的声音。
>
> “我最后问一次。”他拔刀时，左肩比右肩慢了半寸，“人，还是名册？”

### 第一轮：立场

| 选项 | 条件 | 结果 |
|---|---|---|
| 【把人和文书交给他】 | 仍可交付 | 不再战斗，进入《守诺成刃》；根据是否背弃他人生成负面或中性评语 |
| 【把残页抛入火中，只求他放人】 | 持有 `original_leaf` 或文书 | 失去史证；赵衡可能放行阿萝，但要求杜缄留下受罚 |
| 【拔剑，为身后的人争一条路】 | 无 | 进入第二轮；赵衡关系决裂 |
| 【先与他联手挡住趁乱劫掠者】 | D08 未直接决裂；城门已乱 | 两人共同作战一轮；之后仍需解决阿萝和文书归属 |
| 【指出他的左肩旧伤】 | 已完成任朔相关见闻 `zhao_shoulder_known` | 赵衡戒备上升，但其第一轮攻势减弱 | 解锁“不靠数值的新周目优势”，不能直接获胜 |

### 第二轮：用本局所学破局

| 选项 | 条件 | 代价与结果 |
|---|---|---|
| 【横剑截流，以守为攻】 | `guard_art` | 内息 -1；拖住赵衡，让身后队伍通过；杜缄伤势至少重伤 |
| 【踏墙换位，越过刀锋去开侧门】 | `roam_art`；未穿札甲 | 内息 -1；不必击败赵衡，但短剑可能遗落；队伍获得出路 |
| 【以师门旧势诱他卸兵】 | `manual_half` 或曾接受赵衡指点 | 若短剑完整，可解锁 `disarm_art`；失败则自己失去兵器 |
| 【用札甲硬受一刀，贴身反击】 | 穿破损札甲 | 札甲损坏；可以重创赵衡，但风声已无意义地达到缉拿 |
| 【弃剑抱住他，一同滚下石阶】 | 无 | 双方重伤；不杀路线仍成立，但杜缄可能无法出城 |

### 第三轮：如何结束

| 选项 | 条件 | 结果方向 |
|---|---|---|
| 【杀死赵衡】 | 仍持可用兵器；赵衡已露破绽 | 主角存活率提高；关闭“不杀而胜”；师门传承断裂或由杜缄独占 |
| 【卸下他的刀，不取性命】 | `disarm_art` 或守剑路线满足特殊条件 | 获得“不杀而胜”；赵衡可能醒悟断后，也可能带伤失踪 |
| 【折断自己的剑，断绝同门】 | 师门短剑 | 赵衡停手片刻；失去兵器与部分武学传承，开启“弃刃归隐” |
| 【把《折锋六势》还给他】 | `manual_half`；赵衡未杀害关键人物 | 以传承换放行；杜缄本局不保留进阶武学，但人物关系出现特殊结局 |
| 【趁城门大乱离开】 | 游剑路线或余时尚存 | 不分胜负；赵衡存活并成为后续篇章伏笔 |

## D12 留人、留剑或留名

**地点**：长安城外或城门废墟  
**目的**：最终价值选择与结算，不再增加战斗。

### 场景文本

> 入夜之后，长安方向的火照亮了半边天。你身边剩下的人、手里的剑和怀中的残页，没有一样还是出发时的模样。
>
> 远处有人催你继续赶路。你知道自己只能把一件东西完整地带进后世。

### 选项

| 选项 | 主要条件 | 结局倾向 |
|---|---|---|
| 【先安置活下来的人】 | 有阿萝、许简或军属生还 | 《负刃护人》；真相和武谱可能不完整，人物生还优先 |
| 【整理底稿，记下发生的一切】 | 有残页、文书或许简 | 《残剑留录》；必须接受至少一名关键人物未能获救 |
| 【将残谱与兵器托付后辈】 | 有秘籍或师门短剑 | 武学传承结局；杜缄本人可能无名或失去武功 |
| 【回望长安，独自离去】 | 无强制条件 | 根据履约、杀人、人物命运生成《守诺成刃》或失败变体 |

# 5. 结局判定

## 5.1 结局族

| 结局族 | 优先条件 | 必付代价 |
|---|---|---|
| 《守诺成刃》 | 完成赵衡或王允府命令，或独自带文书出城 | 阿萝、军属、许简中的至少一方受损；杜缄被评价为“守令而少问” |
| 《负刃护人》 | 阿萝或军属多数生还，且杜缄主动背离命令 | 失去身份、师门关系、史证或出城机会中的至少一项 |
| 《残剑留录》 | 原始残页或可靠记录保存，且有人将其带出 | 无法同时救下全部关键人物；杜缄自身伤势或名声承担代价 |
| 《城下无名》 | 重伤且无人接应，或所有托付同时失败 | 仍保留局部人物命运和下一周目线索，不显示“游戏失败” |

## 5.2 武学与行为标签

- `短剑守义`：守剑路线完成护送；
- `一人当巷`：以横剑截流让整队通过；
- `不杀而胜`：终局击败或摆脱赵衡且未杀人；
- `负伤断后`：重伤状态下完成 D10～D11；
- `残谱传人`：秘籍或兵器成功托付；
- `弃刃归隐`：折断或放弃师门兵器后离开；
- `只守一诺`：完成原命令但放弃所有其他人物；
- `剑下有愧`：杀死已失去战意或可以放过的对手。

# 6. 路线平衡检查

## 6.1 不允许出现的完美路线

一局不能同时满足：

- 阿萝、许简、全部军属和杜缄均安全；
- 原始残页与全部秘籍完整保存；
- 赵衡得到宽恕且不再造成任何伤害；
- 杜缄保持无伤、低风声和王允府身份；
- 守剑与游剑全部成招。

如果测试中出现以上组合，应增加时间、装备槽位或关系代价，而不是简单提高数值门槛。

## 6.2 路线价值对称

- 守剑更容易保人，但更容易负伤和错过时间；
- 游剑更容易控场和脱身，但必须信任阿萝或他人能暂时自保；
- 履行命令可以降低过程难度，但人物代价更高；
- 保存证据提供史卷价值，但不应自动获得更高道德评价；
- 放弃秘籍可以换取时间，不能被视为低级玩法。

# 7. 任朔路线预留接口

杜缄路线向任朔路线输出：

| 状态 | 任朔路线变化 |
|---|---|
| 杜缄在 D04B 救下持卷者 | 任朔发现对方活着，并得知“王允府门客未必都在灭口” |
| 杜缄留下破损札甲或止血散 | 任朔可以在军属里巷取得对应装备 |
| 杜缄烧毁文书 | 任朔只得到口头传闻，魏石更容易控制叙事 |
| 杜缄在 D09 割断棚架 | 任朔后续路线受阻，但可发现独特剑痕 |
| 杜缄把剑印交给阿萝 | 任朔可借剑印确认阿萝曾与杜缄同行 |
| 杜缄放过赵衡 | 任朔终局可能遭遇负伤的赵衡 |
| 杜缄杀死赵衡 | 任朔失去赵衡视角，但可以获得其遗留刀谱线索 |

任朔路线反向输入的首个接口为 `zhao_shoulder_known`：玩家完成相关任朔事件后，杜缄 D11 可发现赵衡左肩旧伤，但仍需匹配的武学与行动才能利用。

# 8. 纸面测试记录模板

每次试玩记录：

- 玩家选择的武学路线；
- 玩家在 D03、D05、D08、D10 的犹豫时间；
- 玩家是否理解每项选择的主要代价；
- 玩家获得招式后是否在 D09 或 D11 主动使用；
- 玩家最在意的人物；
- 玩家对赵衡的评价是否发生变化；
- 玩家是否认为存在明显正确答案；
- 玩家结算后最想尝试的新路线；
- 玩家是否愿意进入任朔视角。
