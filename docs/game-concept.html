三国乱世为骨 · 江湖武侠为魂 · 纯文字探索 RPG / 类 MUD

从无名之人出发，在城镇、荒道、客栈、废庙与军阀势力之间游历，通过探险、练功、选择与结交人物，卷入乱世背后的江湖纷争。

二核心设计目标

历史 + 武侠

不复刻演义，不架空玄幻，让玩家从江湖底层看到汉末乱世。

文字为主

用房间描述、传闻、对话与战斗文字构建沉浸体验。

剧情驱动

任务与人物关系是玩家继续探索的主要动力，而非单纯引导练级。

MUD 感

房间、出口、NPC、物品、技能、战斗、声望与指令交互俱全。

低门槛操作

吸收 MUD 的系统深度，避免过高的指令记忆成本。

Web 优先

先做浏览器原型，预留移动端、多语言与未来独立发行。

四像对话一样进入世界
去北边那条竹林小路。
问掌柜有没有见过受伤的剑客。
我想找个安静地方练一会儿吐纳。
如果那山贼拔刀，我先后退半步，用剑护住要害。
你想这么做，但眼下并没有合适的机会。也许可以先调查屋内的人，或询问掌柜。

失败反馈也要有叙事感 —— 让玩家知道"为什么不行"和"接下来可以试什么"。

五深度来自选择与系统联动

武功路线

不同武功带来战斗与剧情的差异化体验。

声望 · 阵营 · 关系

人物关系与立场影响任务结果走向。

隐藏线索

探索发现的细节会改变玩家对事件的判断。

多解法

同一问题可用武力、交涉、潜行、医术、声望等方式解决。
