八文案语气 · 宜 / 忌

宜

  • 「你想这么做，但眼下没有合适的机会。也许可以先调查屋内的人。」
  • 失败反馈给出"为什么不行"+"可以试什么"
  • 保持武侠/古白话口吻，融入世界
  • 文案走文本 key，预留国际化

忌

  • 「无效指令」「Error: invalid action」
  • 生硬报错、跳出世界感
  • 现代网络用语 / 表情滥用
  • 大面积铺强调色，破坏沉稳基调
