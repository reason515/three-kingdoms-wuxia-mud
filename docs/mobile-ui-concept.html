这版在试什么

一屏一局场景、状态、成长与选择处于同一注意力流，不像缩小的桌面面板。
文字是画面水墨只建立空间轮廓，真正的动作、距离与危险仍由文字完成。
成长可见领悟进度紧贴剧情，学会的不是“+5”，而是下一幕的新行动。
选择有代价每个按钮直接呈现意图和风险，但不提前剧透完整后果。
可点击行动选项与底部“人物 / 武学 / 行囊 / 史卷”体验状态反馈。
