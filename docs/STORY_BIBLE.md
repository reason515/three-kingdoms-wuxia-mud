# 1. 文档目的

本文定义《长安残照》的历史边界、武侠主线、多人物视角、角色弧光、武学成长、秘籍装备和结局方向，作为事件流程、系统设计、文本写作与数据建模的共同依据。

v0.2 的核心调整是：

- 将“查明清籍案”从主线降为历史背景和多周目线索；
- 将护送、追杀、师门恩怨、同袍之义和武学突破提升为主要体验；
- 让成长、秘籍和装备直接改变剧情选择；
- 让玩家首先想成为不同的侠客，其次才是补全事件真相。

所有原创姓名、组织、武学与物品均为工作设定。

# 2. 篇章核心

## 2.1 一句话梗概

董卓死后，长安人心惶惶。王允的门客、吕布的并州旧卒、被清算的军属和趁乱而起的游侠都在争夺一条出城生路。玩家分别扮演杜缄与任朔，在城破前护送、追击、习武和决斗，并决定自己究竟为命令、同袍还是无辜者拔刀。

## 2.2 核心命题

> 当你所忠之人正在作恶，守义是继续守诺，还是拔剑阻止他？

杜缄面对“受托之义”，任朔面对“同袍之义”。两人都必须发现：忠诚不能免除选择，也不能替持剑之人承担后果。

## 2.3 体验优先级

1. 武学路线与关键交锋；
2. 人物命运与江湖恩义；
3. 多人物视角下的历史真相；
4. 秘籍、装备、称号和特殊挑战；
5. 排行榜与长期竞争。

# 3. 多周目叙事规则

## 3.1 史卷结构

局外玩家是《汉末江湖录》的匿名编纂者。每个周目扮演同一历史事件中的一名固定原创人物。

周目不是时间轮回，而是：

- 不同人物留下的经历；
- 同一事件的不同立场；
- 不同武学道路形成的人生可能；
- 编纂者依据史证进行的有限重构。

## 3.2 多人物交叉

两名主角的路线必须产生：

- **行动交叉**：一人的选择改变另一人的场景；
- **武学交叉**：一人发现的破绽、招式或传承成为另一人的解锁条件；
- **人物交叉**：同一人物在双方眼中呈现不同面貌；
- **证词交叉**：后续周目纠正上一周目的误解。

## 3.3 记录可信度

| 类型 | 含义 |
|---|---|
| 已确认史实 | 构成不可轻易改变的历史锚点 |
| 人物亲历 | 角色亲眼所见，但可能误解原因 |
| 人物证词 | 可能隐瞒、夸大或说谎 |
| 江湖传闻 | 可以提示路线，不直接视为事实 |
| 编纂者推定 | 玩家根据多周目材料形成的解释 |

真相用于增加判断深度，不把游戏变成逐项搜证的破案流程。

# 4. 历史范围

## 4.1 时间与地点

- 时间：东汉初平三年（192 年），董卓被诛后至李傕、郭汜等攻陷长安；
- 地点：长安城内及近郊；
- 历史跨度：约一个多月；
- 主要可玩时间：围城前后的最后数日；
- 单局目标时长：15～25 分钟。

## 4.2 不可改变的历史锚点

1. 王允联合吕布等人诛杀董卓；
2. 李傕、郭汜、樊稠、张济等董卓旧部起兵反攻；
3. 贾诩曾劝其集众攻取长安；
4. 王允一方迎战失利，徐荣战死，胡轸归属发生变化；
5. 李傕等最终进入长安；
6. 吕布败走；
7. 王允留在长安并被杀；
8. 汉献帝及朝廷落入李傕、郭汜等控制；
9. 城破伴随杀戮、劫掠与民众伤亡。

## 4.3 玩家可以改变的范围

玩家可以改变：

- 原创人物的生死、关系、武学道路和江湖评价；
- 一批军属、百姓和伤者能否离城；
- 阿萝、许简、赵衡和魏石的命运；
- 某部武学的传承去向；
- 名册及其原始残页是否保存；
- 杜缄与任朔是否互助、误伤、决斗或留下信物；
- 后世如何记录这群无名者。

玩家不能改变长安失守及主要历史人物的既定结局。

## 4.4 待核验史实

正式文本落笔前继续核对：

- 王允拒绝赦免的具体范围与过程；
- 城门被打开的具体经过；
- 围城日数和事件日期；
- 吕布与郭汜交锋的记载差异；
- 汉长安城道路、里坊、军营和城门称谓；
- 当时官职、兵器、甲具、文书与封泥形式。

# 5. 武侠表现层级

MVP 采用**经典中武**：

- 存在内息、吐纳、轻身和明确的高手层次；
- 高手能在巷道、屋脊等环境中应对数名普通军卒；
- 箭弩、重伤、围攻和军阵仍然致命；
- 个人无法凭武力改变长安陷落；
- 武学优势主要用于护人、突围、擒拿、拖延和改变局部命运。

不使用神通、飞行、隔空杀人和摧城级武力。

# 6. 历史背景线：清籍之争

## 6.1 功能定位

“清籍簿”不再是要求玩家侦破的核心谜案，而是迫使各方争夺人物、路线和生存机会的背景装置。

它主要承担：

- 让玩家判断谁值得保护；
- 解释赵衡和魏石为何走向极端；
- 为不同周目提供认知反转；
- 让保存证据与保护活人产生冲突。

## 6.2 底层事实

原始名册曾区分董卓核心旧部、普通军吏军属及可提供证词者。赵衡删去副本上的分类标记，并加入私人仇敌，希望以恐慌迫使董卓旧部现身。

董卓旧部的联络者和魏石又把被改动的副本当作“王允将尽诛凉州人”的证明，用于召集人手、夺取军马和组织出逃。

这项篡改加剧局部恐慌，但不是李傕等反攻的唯一原因，也不能替王允的历史决策免责。

## 6.3 与主线的关系

阿萝曾协助誊写原始名册，记得分类符号，并保留一页底稿。赵衡要控制她，魏石想利用她证明军中传言，许简希望她活着作证。

阿萝不是等待玩家搬运的“任务物品”。她掌握医术、藏匿路线和自己的判断，会拒绝牺牲病患只为保存证据。

玩家真正面对的是：

> 把她交给谁、是否尊重她的选择，以及愿意为护送哪些人付出什么。

# 7. 可玩主角

## 7.1 杜缄：受托之剑

| 项目 | 设定 |
|---|---|
| 身份 | 王允府外围门客兼信使 |
| 年龄 | 23 岁 |
| 出身 | 京兆书吏之家，少年时受太原游侠传授短剑 |
| 师门关系 | 与赵衡出自同一位已故师父，赵衡为其前辈 |
| 初始武学 | 短剑根基、基础吐纳、辨迹 |
| 初始装备 | 师门短剑、便服、剑印 |
| 初始任务 | 护送许简及封存文书前往安全据点，并协助控制阿萝 |
| 初始认知 | 相信强硬处置能够避免长安再乱 |
| 认知盲区 | 不知道赵衡改动副本，也不知道阿萝会被灭口 |
| 人物弧光 | 从“受人之托便应完成”走向判断不义之托是否仍值得守 |

### 杜缄的武学方向

- **守剑**：护人、封路、格挡和拖延；
- **游剑**：轻身、绕行、脱离和潜入；
- **制剑**：卸兵、擒拿和不杀而胜，完整垂直切片再扩展。

MVP 每局最多深入其中一条，并获得另一条的基础招式。

### 杜缄的关键取舍

- 接受赵衡指点，还是质疑前辈；
- 完成许简的托付，还是优先保护阿萝和养母；
- 使用师门剑印换取通行，还是隐藏身份；
- 在终局杀死、放过、废去或联手赵衡。

## 7.2 任朔：同袍之刀

| 项目 | 设定 |
|---|---|
| 身份 | 吕布军中的并州斥候，原为边地刀客 |
| 年龄 | 28 岁 |
| 出身 | 五原军户，擅骑射与环首刀 |
| 初始关系 | 魏石曾在边地救过他的性命 |
| 初始武学 | 军中刀法、骑射、战场判断 |
| 初始装备 | 旧环首刀、破损札甲、同袍箭簇 |
| 初始状态 | 带伤，并被怀疑临阵脱逃 |
| 初始任务 | 找回魏石，确认军属处境并赶在撤军前归队 |
| 初始认知 | 相信朝廷准备无差别清洗外来军人 |
| 认知盲区 | 不知道魏石明知名册可疑，仍借谣言夺取军马 |
| 人物弧光 | 从“不负同袍便是义”走向承认真正的义也可能要求亲手阻止同袍 |

### 任朔的武学方向

- **破阵**：强攻、威慑、快速突围；
- **护军**：替人挡击、稳住阵脚、负伤作战；
- **骑战**：追击、骑射和路线控制，完整垂直切片再扩展。

### 任朔的关键取舍

- 追随魏石，还是阻止他利用军属；
- 回营洗清嫌疑，还是护送伤者；
- 穿札甲正面突围，还是换便服隐藏军人身份；
- 在终局守住同袍之情，还是守住自己认可的义。

# 8. 关键原创人物

## 8.1 赵衡：被复仇吞没的前辈

- 身份：王允府外围游侠首领，杜缄的同门前辈；
- 武学：关中长刀与师门卸兵法，实力高于两名初始主角；
- 公开目标：追捕董卓旧党、控制阿萝和名册；
- 私人目标：为迁都途中死去的家人复仇；
- 秘密：改动名册副本并加入私人仇敌；
- 武侠矛盾：他曾救过杜缄，也会为了“止乱”错杀无辜；
- 终局可能：被杀、被废、醒悟断后、继续追杀或失踪；
- 设计边界：他不是操纵历史的幕后黑手，而是局部悲剧的推动者和武学镜像。

## 8.2 阿萝：不愿被任何一方代表的医者

- 身份：为军属与百姓治伤的医者；
- 目标：带弟弟和病患离开长安；
- 能力：医术、辨伤、熟悉里巷藏匿路线；
- 秘密：记得原名册分类并保留一页底稿；
- 武侠作用：可以救治内伤、指出错误运功，也会拒绝把药只留给高手；
- 选择权：是否出城、是否作证、是否保存残页由关系和局势共同决定。

## 8.3 魏石：救过人也害过人的同袍

- 身份：并州军斥候，任朔的救命恩人；
- 武学：快刀、骑射、伏击；
- 公开目标：救军属出城；
- 私人目标：借混乱夺取军马和财物，为离开吕布后自立做准备；
- 秘密：知道手中副本信息不全，仍主动传播最坏解释；
- 终局可能：与任朔和解、决裂、决斗、被救或被亲手阻止。

## 8.4 许简：相信文书仍能救人的属吏

- 身份：王允府低阶属吏，名册原编者；
- 目标：让分类底稿和阿萝的证词存活；
- 局限：不通武艺，仍低估文书落入强权手中的危险；
- 秘密：保有一部分原始底稿，也知道赵衡改动过副本；
- 作用：让杜缄在“护送命令”和“护送活人”之间发生冲突。

# 9. 武学成长

## 9.1 成长原则

成长不依靠重复刷敌人，而来自：

- 有代价的实战使用；
- 高手指点；
- 失败与负伤；
- 秘籍参悟；
- 为保护某人突破原有用法。

成长优先解锁新的行动，不只增加数值。

## 9.2 每局成长节奏

每名主角拥有两个成长节点：

1. **定路**：第一次重要交锋后选择本局武学倾向；
2. **成招**：通过指点、秘籍或实战领悟一个进阶招式。

终局必须提供至少一个由本局所学解锁的特殊行动。

## 9.3 实战领悟

只有符合武学主题的行动才能积累领悟。例如杜缄的守剑领悟来自：

- 替他人挡下攻击；
- 守住狭窄通道；
- 以防守争取交谈或撤离时间。

任朔的护军领悟来自：

- 带伤保护同伴；
- 放弃追击以稳住阵脚；
- 在撤退中保持队形。

同一场低风险战斗不能反复刷取领悟。

# 10. 秘籍设计

## 10.1 杜缄：《折锋六势》残谱

- 载体：师父留下的残缺剑谱，后半部分在赵衡手中；
- 方向一：**横剑截流**，强化守路与护人；
- 方向二：**卸锋**，格挡后尝试夺兵；
- 获取代价：接受赵衡指点、夺取后半残页，或在决斗中自行悟出其一；
- 剧情作用：是否获得完整传承与如何对待赵衡绑定。

## 10.2 任朔：《并州军刀撮要》

- 载体：军中老卒记录的刀法与撤阵心得；
- 方向一：**破围**，消耗内息快速突破多人封锁；
- 方向二：**回锋护列**，放弃追击以保护身边目标；
- 获取代价：回营寻找旧物、从魏石处取回，或以救人换取老卒口授；
- 剧情作用：选择“破阵”还是“护军”，也对应任朔对同袍之义的理解。

## 10.3 共享残篇：《调息图诀》

- 功能：压制伤势、闭息藏身或在终局多支撑一轮；
- 限制：强行运功会在结算留下内伤；
- 多周目作用：一名主角发现其下落，另一名主角才可能在特定场景取得；
- MVP 可以只实现一个完整用途，其他用途作为后续扩展。

# 11. 装备设计

## 11.1 槽位

MVP 每名角色仅保留：

- 兵器 × 1；
- 衣甲 × 1；
- 信物 × 1；
- 消耗品 × 2；
- 剧情物品独立记录。

## 11.2 关键装备

| 装备 | 优势 | 代价或限制 |
|---|---|---|
| 师门短剑 | 隐蔽、适合巷战、启用杜缄剑招 | 面对甲士与长兵器不利 |
| 旧环首刀 | 强攻、威慑、可破坏障碍 | 难隐藏，使风声趋紧并暴露军人身份 |
| 并州角弓 | 远程压制、射断绳索或火把 | 箭矢有限，近战无法使用 |
| 破损札甲 | 抵挡一次重击或箭伤 | 妨碍轻身，容易暴露身份 |
| 百姓夹衣 | 缓和风声，便于混行 | 几乎没有防护，难藏长兵器 |
| 师门剑印 | 获得同门回应或特殊对话 | 暴露杜缄与王允门客的联系 |
| 同袍箭簇 | 证明任朔与魏石旧谊 | 也会让军中人物认出其身份 |

## 11.3 装备选择

每件重要装备至少解锁一种行动并带来一种代价。装备可以：

- 损坏；
- 被交换；
- 被赠予；
- 用于卡门、断后或证明身份；
- 随人物命运进入结局。

MVP 不设置随机品质、强化、锻造、套装和大量掉落。

# 12. 场景与交锋结构

## 12.1 核心地点

1. **王允府外巷**：受命、师门压力、杜缄起点；
2. **西市废仓**：截信、首次交锋、路线交叉；
3. **军属里巷**：阿萝、病患、装备取舍和护送目标；
4. **并州军营外缘**：任朔的武学与同袍路线；
5. **废寺或旧宅**：秘籍、指点与时间代价；
6. **西侧城门附近**：追杀、城破压力与终局决斗。

地点精确名称待历史地理核验。

## 12.2 单局五幕

1. **受命或寻人**：建立偏见、任务、初始武学和装备；
2. **第一次交锋**：通过行动确定武学倾向；
3. **护送与取舍**：决定救谁、带什么、放弃什么；
4. **学招与决裂**：在秘籍、指点、时间和关系间选择；
5. **城破终局**：用本局构筑面对赵衡、魏石或乱军。

## 12.3 战斗数量

每局目标：

- 2 次普通交锋，每次 1～3 轮；
- 1 次关键决斗或突围，约 3～5 轮；
- 至少一次允许不杀、撤退、护人或以装备解围；
- 战斗结果持续影响伤势、时间、风声、关系和结局。

# 13. 两条路线的交叉

## 13.1 西市废仓

杜缄是否追击魏石、先救伤者或保护文书，会改变任朔随后看到的现场、可取得装备及魏石的伤势。

## 13.2 阿萝与军属

一名主角提前转移病患，另一名主角可能失去史证，却获得更安全的出城路线。公开阿萝的证词会提高真相完整度，也会暴露藏匿点。

## 13.3 武学破绽

任朔可以从交锋中发现赵衡左肩旧伤；杜缄完成任朔路线后可解锁观察选项，但仍需相应剑法才能利用。杜缄留下的剑痕也能帮助任朔判断赵衡的卸兵方式。

## 13.4 城门终局

一人留下的兵器、信物、假路引或被毁的通道，会成为另一周目的条件。两名主角可以短暂相遇、误战或共同断后，但不会立即交换全部真相。

# 14. 结局族

所有结局均发生在长安失守的大势下，并根据主角、武学、装备和人物命运生成变体。

## 14.1 《守诺成刃》

- 完成所属阵营或重要人物的托付；
- 以牺牲另一段关系或一批人的安全为代价；
- 武学得到认可，但主角必须承担“只守承诺、不问是非”的评价。

## 14.2 《负刃护人》

- 背弃命令或同袍，护送阿萝、病患或军属离开；
- 失去身份、名声、秘籍或归队机会；
- 真相可能散失，但有人因主角的武艺和选择活下来。

## 14.3 《残剑留录》

- 保存关键史证或武学传承并托付后人；
- 必须失去任务、兵器、关系或身体完好中的至少一项；
- 无法同时救下所有关键人物；
- 为后续执笔定史和传承路线提供条件。

## 14.4 《囚剑长安》

- 主角主动以自由换取他人通行，或落网后无法脱逃；
- 兵器、秘籍和行动权可能被夺走；
- 主角仍可通过供词、沉默、交换或狱中留字影响人物命运；
- 城破可能令其脱逃、失踪、被迫效力或死于狱中；
- 被捕不是自动失败，也不能成为比战斗更安全的无代价路线。

## 14.5 生死与自由维度

每个结局族另行记录：

- 生死：生还、死亡、生死不明；
- 自由：自由、落网、收押、脱逃、被迫效力；被追捕是缉拿状态下的处境，不是失去自由；
- 风声达到缉拿只代表公开搜捕，被敌方控制后才算落网；
- 死亡必须有明确风险预兆，或来自玩家主动舍命与濒危后再次受创；
- 落网后最多增加一个短节点，处理供词、交换、越狱或主动受押，再进入结算。

## 14.6 武学结局标签

在结局族之外增加标签：

- 不杀而胜；
- 一人当巷；
- 并州孤刀；
- 短剑守义；
- 负伤断后；
- 残谱传人；
- 弃刃归隐；
- 解剑护人；
- 城破脱枷；
- 狱壁留招。

# 15. 重玩动力

玩家重新进入《长安残照》的主要理由是：

1. 尝试另一条武学路线；
2. 改变赵衡、魏石、阿萝和许简的命运；
3. 解锁此前不可用的战斗与护送行动；
4. 发现另一主角留下的武学破绽和信物；
5. 收集秘籍残篇、装备经历和武学结局；
6. 补全清籍争议背后的真相；
7. 完成不杀、低伤、基础武功等高难挑战。

# 16. MVP 验证标准

纸面或 Web 测试需要回答：

1. 玩家是否首先把它描述为武侠故事，而不是破案游戏？
2. 玩家是否感到能力在一局内发生了可见成长？
3. 新招式是否改变了后续行动，而非只增加成功率？
4. 玩家是否因练功、护人、赶路和保存装备之间的冲突而犹豫？
5. 不同武学路线是否产生不同终局打法？
6. 玩家是否愿意为了另一套武学或人物命运立即重开？
7. 第二人物视角是否揭示新事实，但不压过江湖主线？
8. 文字战斗是否清晰呈现距离、环境、招式与局势？
9. 玩家是否理解缉拿、落网和收押的区别？
10. 死亡或被捕结局是否有充分预兆，并仍然保留有意义的人物结果？
11. 是否存在明显无条件最优的装备或成长路线？

# 17. 史料与原创声明

正式写作优先核对：

- 《后汉书》相关列传；
- 《三国志·魏书·吕布张邈臧洪传》及裴松之注；
- 《资治通鉴》卷六十；
- 汉长安城历史地理、兵器、军制、衣甲和文书资料。

以下均为原创设定，不应被误认为史实：

- 清籍簿及其篡改；
- 杜缄、任朔、赵衡、阿萝、魏石、许简；
- 《折锋六势》《并州军刀撮要》《调息图诀》；
- 两名主角对长安城破过程的具体介入。
