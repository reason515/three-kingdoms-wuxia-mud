# 1. 文档目的

本文定义《长安残照》的历史边界、底层真相、多人物视角、关键角色、史证分配和 MVP-0 结局，作为后续事件流程、系统设计、文本写作与数据建模的共同依据。

当前版本首先服务于纸面原型，不追求一次写完整篇章。所有原创姓名与细节均为工作版本。

# 2. 多周目叙事规则

## 2.1 玩家身份

局外玩家是《汉末江湖录》的匿名编纂者。每次进入篇章，玩家读取并重构一名无名者在同一历史事件中的经历。

周目不是时间轮回。不同周目代表：

- 不同人物留下的记录；
- 同一事件的不同立场；
- 史证支持下的不同可能过程；
- 编纂者对缺失记录的有限重构。

## 2.2 记录不等于事实

游戏区分四类信息：

| 类型 | 含义 |
|---|---|
| 已确认史实 | 多种可靠来源能够互证，构成不可轻易改变的历史锚点 |
| 人物证词 | 某人相信或声称发生过的事，可能说谎或误判 |
| 江湖传闻 | 来源不明但可能包含线索，不直接视为事实 |
| 玩家推定 | 编纂者根据已获史证作出的解释，可被后续周目修正 |

游戏不能因为玩家完成一次路线，就把该路线中的所有叙述自动标为真相。

## 2.3 多人物交叉原则

两名主角的经历必须至少形成三类交叉：

1. **行动交叉**：一人的选择改变另一人随后遇到的局面；
2. **证词交叉**：一人对事件的理解，被另一人的史证支持或推翻；
3. **人物交叉**：同一 NPC 在两名主角面前呈现不同身份、目的或态度。

# 3. 历史范围

## 3.1 时间与地点

- 时间：东汉初平三年（192 年），董卓被诛后至李傕、郭汜等攻陷长安；
- 核心地点：长安城内及近郊；
- 篇章跨度：约一个多月，以围城前后的最后数日作为主要可玩时间；
- MVP-0 单局体验：10～15 分钟。

## 3.2 已确认的历史锚点

以下内容原则上不可由玩家改变：

1. 王允联合吕布等人诛杀董卓；
2. 董卓旧部李傕、郭汜、樊稠、张济等在未获赦免后起兵；
3. 贾诩曾劝其集众反攻，而非解散逃亡；
4. 王允一方派兵迎战，徐荣战死，胡轸转向或归附反攻军；
5. 李傕等围攻并最终进入长安；
6. 吕布败走长安；
7. 王允拒绝随吕布出逃，最终被杀；
8. 汉献帝及朝廷落入李傕、郭汜等控制；
9. 城破伴随严重杀戮、劫掠与民众伤亡。

## 3.3 需要谨慎处理的史料差异

以下细节在正式剧情定稿前需要继续核对不同史料：

- 王允拒绝赦免的具体范围、措辞和决策过程；
- 城门从内部被打开的具体人员与经过；
- 吕布与郭汜交锋的具体形式；
- 围城日数与各事件的精确日期换算；
- 部分将领在反攻过程中的归属变化；
- 城破后的具体伤亡数字。

MVP 不把有争议的细节写成唯一确定事实，可以将其转化为互相矛盾的证词。

## 3.4 玩家可以改变的范围

玩家可以改变：

- 原创人物的生死、关系、归属和历史评价；
- 一份民间名册及其副本是否保存；
- 少量百姓、军属和证人能否离城；
- 某条谣言是否扩散；
- 某封信以何种版本到达何人手中；
- 后世能否知道名册的原始用途；
- 两名主角是否相遇、互助、误伤或彼此误解。

玩家不能改变：

- 长安最终失守；
- 王允和吕布阵营的总体失败；
- 李傕、郭汜等控制朝廷；
- 已明确列入历史锚点的历史人物结局。

# 4. 核心虚构事件：清籍案

## 4.1 表层说法

董卓死后，长安流传着一份“清籍簿”。不同人对它有不同解释：

- 王允府外线称其用于查明董卓死党；
- 凉州军属认为它是准备株连清算的杀人簿；
- 市井传闻称名单上的人会在三日内被捕；
- 有人声称名册可以证明自己只是被裹挟的普通军属。

## 4.2 底层真相

“清籍簿”最初并非单纯的诛杀名单，而是一套尚未执行的分类名册：

- 第一类：直接参与董卓暴政和军政核心者；
- 第二类：普通军吏、被征发者及其军属；
- 第三类：能够提供证词、换取自保的人。

名册原本可能用于有限甄别、登记与安置，但在王允拒绝赦免董卓核心旧部的政治环境下，它同时具备清算工具的危险性。

一名王允府属吏私自改动了副本：删除分类标记，让整份名册看上去像无差别缉捕名单。他希望利用恐慌迫使军属交出藏匿的董卓亲信，也借机清除私人仇敌。

与此同时，董卓旧部的联络者主动传播“王允将尽诛凉州人”的说法，把被篡改副本当作动员证据。

因此：

- 王允确实作出了拒绝赦免核心旧部的历史决定；
- 恐慌并非完全凭空捏造；
- 篡改名册加剧了恐慌，但不是李傕等反攻的唯一原因；
- 朝廷强硬派和董卓旧部都在利用普通人的恐惧；
- 没有任何一方掌握全部真相，也没有一方完全无辜。

## 4.3 核心道德困境

保存名册，可能为普通军属留下申辩证据，也可能让名单落入清算者手中。

毁掉名册，可以立刻保护被记录者，却会令原始分类和篡改事实永久消失。

公开名册，可能阻止谣言，也可能在失控的长安城中迅速变成新的杀人依据。

> 名册不是善恶道具。它的意义由持有者、时机、版本和公开方式共同决定。

# 5. 可玩主角

## 5.1 主角一：杜缄（工作名）

| 项目 | 设定 |
|---|---|
| 身份 | 王允府属吏许简的外线信使 |
| 年龄 | 23 岁 |
| 出身 | 京兆普通书吏之家，幼时随游侠师父学过短剑与辨迹 |
| 初始专长 | 谍术、辩才 |
| 弱项 | 正面战斗、军中威望 |
| 初始任务 | 将封存副本和口信送至城西接头人手中 |
| 初始认知 | 相信名册是稳定长安、区分首恶与被迫从者的必要工具 |
| 认知盲区 | 不知道自己携带的副本已被删去分类标记 |
| 私人牵挂 | 养母居住在凉州军属聚居的里巷，名字可能出现在名册上 |
| 人物弧光 | 从“只负责送达，不负责内容”走向决定自己是否为所传之令承担责任 |

### 杜缄的核心取舍

- 遵从命令，还是拆封确认；
- 保住政治信任，还是警告军属；
- 保存证据，还是保护养母；
- 承认自己看过名册，还是继续以无知自保。

## 5.2 主角二：任朔（工作名）

| 项目 | 设定 |
|---|---|
| 身份 | 吕布军中的并州斥候 |
| 年龄 | 28 岁 |
| 出身 | 五原边地军户，擅骑射和近身短兵 |
| 初始专长 | 轻功、兵法 |
| 弱项 | 朝堂人脉、文书辨伪 |
| 初始状态 | 带伤，疑似被己方误认为逃兵 |
| 初始任务 | 找回失踪同袍，并确认城西军属是否即将遭到清算 |
| 初始认知 | 相信王允准备清洗所有董卓旧部及外来军人，并州军只是下一个目标 |
| 认知盲区 | 不知道传播消息的人故意混淆了凉州核心旧部、普通军属与并州军 |
| 私人牵挂 | 失踪同袍曾救过他的命，却可能参与抢夺名册 |
| 人物弧光 | 从“只信同袍，不信纸上之言”走向判断共同经历是否足以替代事实 |

### 任朔的核心取舍

- 救失踪同袍，还是阻止他传播假名册；
- 保护并州军声誉，还是承认内部有人趁乱劫掠；
- 护送军属出城，还是赶回军营备战；
- 相信亲历者的证词，还是相信来历可疑的文书。

# 6. 关键原创人物

## 6.1 许简：名册原编者

- 身份：王允府低阶属吏，杜缄的直接联系人；
- 公开目标：完成清籍，稳定长安；
- 私人目标：证明朝廷仍能区分首恶与被裹挟者；
- 秘密：他保留了一页带分类标记的原始底稿；
- 代价：若公开承认名册被篡改，他本人会先因失职被治罪；
- 双视角差异：杜缄视其为谨慎师长，任朔最初只知道他是“造杀人簿的人”。

## 6.2 赵衡：强硬派执行者

- 身份：王允府外围武吏；
- 公开目标：追查董卓死党；
- 私人目标：借清算为死于迁都途中的家人复仇；
- 秘密：他删除了副本上的分类标记，并加入私人仇敌；
- 自我辩护：乱世无法逐一审辨，宁可错捕，不可放过；
- 边界：他不是操纵全部历史的幕后黑手，只是让局部悲剧恶化的人。

## 6.3 阿萝：军属医者

- 身份：在城西里巷为军属和百姓治伤的年轻医者；
- 公开目标：带年幼弟弟离开长安；
- 私人目标：找到被征为向导后失踪的父亲；
- 秘密：她曾替许简誊写部分原始名册，记得分类符号；
- 双视角作用：她是杜缄的养母邻人，也是任朔寻找同袍时的救命人；
- 代价：她的证词能证明名册被改动，却会暴露整条军属藏匿路线。

## 6.4 魏石：任朔的失踪同袍

- 身份：并州军斥候；
- 公开目标：警告军属，组织逃离；
- 私人目标：利用混乱带走曾受董卓旧部控制的一批军马和财物；
- 秘密：他抢到的是被篡改副本，并明知其中部分内容可疑；
- 复杂性：他确实救过人，也确实准备借谣言牟利；
- 双视角作用：杜缄视其为截信者，任朔视其为生死同袍。

# 7. 历史人物使用边界

## 7.1 王允

- 主要通过命令、属吏言谈和结果出现；
- 不能将拒绝赦免完全归咎于虚构人物欺骗；
- 表现其忠于汉室、刚决自负及政治判断的代价；
- MVP 不安排玩家直接说服王允改变历史决策。

## 7.2 吕布

- 主要通过军令、军中传闻、远处交锋和败走结果出现；
- 保留其骑射勇武、并州背景和政治处境；
- 不以《三国演义》的貂蝉情节作为史实核心；
- 玩家无法成为左右吕布所有决定的亲信。

## 7.3 李傕、郭汜、贾诩等

- 主要作为城外压力和情报来源；
- 贾诩劝李傕等集众反攻属于历史锚点；
- 不把反攻解释为清籍案单独造成；
- MVP 不安排主角潜入其最高决策层。

# 8. MVP-0 场景范围

MVP-0 暂时只使用五个抽象地点：

1. **王允府外巷**：命令、监视与政治压力；
2. **西市废仓**：名册交易、截信与第一次交锋；
3. **军属里巷**：百姓、军属、阿萝与逃生路线；
4. **并州军营外缘**：任朔的身份危机和军中分歧；
5. **西侧城门附近**：终局收束、撤离与城破压力。

具体里坊、道路和城门名称待历史地理核验后再定，现阶段不使用可能失实的精确名称。

# 9. 史证设计

| 编号 | 史证 | 主要来源 | 支持或挑战的认知 |
|---|---|---|---|
| E01 | 带分类符号的原始残页 | 许简路线 | 证明原名册不是无差别杀人簿 |
| E02 | 删除标记的封存副本 | 杜缄初始携带 | 证明流传版本经过处理，但单独无法证明由谁处理 |
| E03 | 阿萝的誊写记忆 | 军属里巷 | 支持 E01，同时暴露军属藏匿路线 |
| E04 | 魏石取得副本的经过 | 任朔路线 | 证明军中有人明知信息不全仍主动传播 |
| E05 | 赵衡的私人增补名单 | 对质或搜查 | 证明有人借政治清算处理私仇 |
| E06 | 城外招募口号的抄录 | 任朔侦察 | 证明反攻方主动放大“尽诛凉州人”的恐惧 |

## 9.1 分配规则

- 单一周目不能获得全部史证；
- 杜缄路线优先获得 E01、E02、E05；
- 任朔路线优先获得 E03、E04、E06；
- 至少完成两个身份各一次，才能确认“名册被改动且被双方利用”；
- 找到史证不等于能够保住史证，携带、公开和托付必须产生不同后果。

# 10. 两条路线的交叉

## 10.1 交叉点一：西市废仓

- 杜缄可以提前转移、藏匿或照常交付副本；
- 该选择决定任朔后来遇到的是魏石、赵衡，还是一处被烧毁的仓库；
- 任朔可以留下救治痕迹，成为杜缄下一周目判断“截信者并非只为杀人”的线索。

## 10.2 交叉点二：阿萝的去留

- 杜缄若警告阿萝，她会提前转移病患，但 E03 更难取得；
- 任朔若优先护送阿萝，则无法及时回营澄清逃兵嫌疑；
- 任一主角若公开她的证词，另一条路线中的军属藏匿点会更早暴露。

## 10.3 交叉点三：城门终局

- 一人留下的信物、假路引或被烧毁的名册，会成为另一人终局可用或缺失的条件；
- 两人可以在少数路线短暂相遇，但不立即互相说明全部身份；
- 玩家完成两个身份后，才能知道自己曾把上一周目的主角误认为敌人。

# 11. MVP-0 三类结局

所有结局均发生在长安失守的大势下。

## 11.1 《奉命无名》

- 主角完成所属阵营的表面任务；
- 关键人物或军属为此承担代价；
- 原始证据没有完整留存；
- 主角在官方或军中记录里得到认可，却仍不知道自己传递了什么。

主题：服从可以换来秩序感，但不能免除责任。

## 11.2 《焚卷存人》

- 主角毁掉主要名册或放弃追查；
- 一批原创人物得以暂时逃生；
- 篡改事实难以被后世确认；
- 谣言继续成为各方解释失败和仇恨的工具。

主题：救下眼前的人，可能意味着永远失去证明真相的机会。

## 11.3 《残页照世》

- 主角保存至少一项关键史证并托付给可信人物；
- 必须牺牲任务、名声、同袍关系或自身安全中的至少一项；
- 不能同时救下全部关键人物；
- 该结局为后续“执笔定史”提供条件，但不是无代价的完美结局。

主题：留下真相本身就是一种昂贵的行动。

# 12. 纸面原型节奏

每个身份采用相同的五段节奏，但事件内容和认知不同：

1. **受命**：获得任务和带偏见的初始信息；
2. **见疑**：第一次发现任务说明与现场不一致；
3. **交叉**：与另一主角的行动痕迹或共同 NPC 相遇；
4. **承诺**：作出不可逆选择，明确放弃一个目标；
5. **城破**：在历史大势下决定带走人、命令还是证据。

每局控制在 8～12 次有效选择，其中重大不可逆选择 2～3 次。

# 13. 文本基调

## 13.1 场景

- 使用一两个具体物象建立长安乱局，如封泥、血水、马蹄、闭门的里巷和被撕下的榜文；
- 避免大段讲解政治背景，让历史通过命令、传闻、饥饿和人物反应进入场景；
- 每段文字都应为气氛、信息或决策服务。

## 13.2 战斗

- 明确距离、障碍、人数、退路和保护目标；
- 不以招式名堆砌代替动作；
- 胜负之外持续改变伤势、时间、疑迹和人物命运；
- 允许撤退、误伤、保护、欺骗和交换，而非只允许击杀。

## 13.3 历史人物

- 不神化，也不以现代价值观简单审判；
- 通过有限接触表现其力量和局限；
- 让玩家感受到自己面对的是有独立意志的历史人物，而不是等待攻略的 NPC。

# 14. MVP 验证标准

纸面测试后，应能够回答：

1. 玩家是否理解两个身份看到的是同一事件的不同部分？
