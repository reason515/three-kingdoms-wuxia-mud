# 游戏命名

## 正式工作名

# 汉末江湖录

## 名称含义

- **汉末**：指向东汉末年、三国乱世的历史背景，比直接叫“三国”更有时代纵深，也更适合从乱世前夜切入。
- **江湖**：强调武侠视角。玩家不是一开始就站在帝王将相的位置，而是从游侠、门派、客栈、密信、恩怨和传闻进入历史。
- **录**：有记录、志书、行记之意，天然贴合纯文字、MUD、日志流和玩家游历体验。

## 传播定位

一句话表达：

> 一款以东汉末年为背景、以江湖武侠为视角的纯文字探索 RPG。

更有宣传感的版本：

> 乱世将起，江湖未远。你不过一介无名游侠，却将在汉末风云中写下自己的江湖一页。
